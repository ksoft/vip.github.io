## 简介
Arthas 是 Alibaba 在 2018 年 9 月开源的 Java 诊断工具。

## 下载和使用
1、authas是一个jar包，可以直接下载后运行
```markdown
wget https://alibaba.github.io/arthas/arthas-boot.jar

java -jar arthas-boot.jar
```
2、启动后，authas会自动检测存在的java进程，这时候需要选择你想要诊断的进程，回车即可。如下图所示：
